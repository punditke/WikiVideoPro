# WikiVideo Pro 🎥

A premium video discovery portal for Wikimedia Commons content. Built with Next.js, Tailwind CSS, and Framer Motion.

![WikiVideo Pro](public/og-image.png)

## ✨ Features

- **Dark Luxury Design** - Deep zinc and black palette with emerald accents
- **Responsive Grid** - Perfect viewing on mobile, tablet, and desktop
- **Cinematic Theater** - Split-view player with detailed metadata
- **Real-time Search** - Glassmorphism search with suggestions
- **Smooth Animations** - Framer Motion powered transitions
- **Wikimedia Integration** - Direct API access to millions of videos
